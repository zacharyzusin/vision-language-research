# Documentation

This directory contains documentation for the Mixture-of-Prompts CLIP project.

## Setup Guides
- `DATASET_DOWNLOAD.md` - Instructions for downloading datasets
- `KIKIBOUBA_SETUP.md` - KikiBouba dataset setup guide
- `STANFORD_CARS_SETUP.md` - Stanford Cars dataset setup guide

## Analysis Documents
- `K_ANALYSIS.md` - Analysis of K (number of sub-prompts) effectiveness
- `MODE_COLLAPSE_ANALYSIS.md` - Analysis of mode collapse issues
- `MODE_COLLAPSE_FIX.md` - Documentation of mode collapse fixes
- `DIAGNOSTIC_MODE_COLLAPSE.md` - Diagnostic procedures for mode collapse

## Training Guides
- `SUBSET_TRAINING.md` - Guide for training on dataset subsets
- `SWEEP_SETUP.md` - Weights & Biases sweep configuration guide
- `WRASSE_ABLATIONS.md` - Ablation study results for wrasse subset
